1- Caso queira apenas trabalhar no Banco de Dados ... Procure o arquivo 
  
  lili_backup.sql, e copie para sua máquina.

2- Criar o banco de dados na nova máquina

entrar no MySQL cli:

mysql  -u 'seu_usuario' -p [enter do teclado]

digite o seguinte comando:

CREATE DATABASE lili CHARACTER SET utf8mb4 COLLATE utf8mb4_0900_ai_ci;

exit;

explicação do comando:

CREATE DATABASE lili, você esta efetivamente criando um novo banco de nome lili. 

CHARACTER SET utf8mb4 COLLATE utf8mb4_0900_ai_ci; esta defininido as regras referentes
a codificação dos caracteres.

exit; sai do banco de dados.

3- Importar o backup

No terminal:

mysql -u root -p lili < lili_backup.sql


  Isso vai recriar todas as tabelas e inserir os dados. E "voalaaa" está feito agora é só brincar
com o banco de dados. 

4- Para testar se tudo deu certo:

mysql  -u 'seu_usuario' -p [enter do teclado]

* Normalização: É um processo de análise e adequação dos esquemas de tabelas com o objetivo de 
minimizar redundâncias e anomalias de inserção, exclusão e alteração.

** Segunda Forma Normal (2NF): é baseada no conceito de atributo primo e dependência funcional 
total.
• A dependência funcional X Y é uma dependência funcional total se ao remover qualquer 
atributo A de X a dependência funcional deixa de valer.
• Ou seja, para qualquer A ∈ X, (X- A) não determina funcionalmente Y.
